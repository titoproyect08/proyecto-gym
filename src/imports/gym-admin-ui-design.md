Diseña las interfaces completas de un PANEL DE ADMINISTRACIÓN para una aplicación de gimnasio y fitness.

IMPORTANTE:
- La aplicación debe comenzar siempre con una pantalla de inicio de sesión de administrador.
- Solo se deben generar interfaces visuales (UI).
- No se debe almacenar información real.
- No se debe conectar con bases de datos.
- Todos los botones deben tener una función visual dentro de la interfaz.
- Las acciones deben ser simuladas (modo demo).
- Los formularios deben existir, pero solo como interfaz sin guardar datos.

OBJETIVO:
Crear un panel administrativo completo para gestionar todos los aspectos de una aplicación de gimnasio.

ESTILO DE DISEÑO:
- Panel administrativo moderno
- Estilo dashboard profesional
- Diseño limpio
- Modo oscuro preferido
- Navegación lateral (sidebar)
- Iconos para cada módulo
- Tarjetas de estadísticas
- Tablas administrativas
- Formularios claros
- Botones visibles para cada acción

-----------------------------------------------------

PANTALLA 1 — LOGIN ADMINISTRADOR

Elementos:
- Logo de la app
- Campo: correo electrónico
- Campo: contraseña
- Checkbox: recordar sesión

Botones:
- Iniciar sesión
- Recuperar contraseña

Nota:
El botón "Iniciar sesión" solo debe llevar al dashboard (modo simulación).

-----------------------------------------------------

PANTALLA 2 — DASHBOARD ADMIN

Elementos:
Tarjetas de estadísticas (datos simulados):

Secciones:
- Actividad reciente

- Rutinas populares

Botones:
